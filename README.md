# #ros1-practice
ROS1 Melodic or Noetic+Python3 package for sessions  

## Files made by
Koki Nagahama (@hamstick)

## Usage
catkinワークスペースができている状態で，そこ以外のお好きなディレクトリで
```
$ git clone https://github.com/enjoydolylab/ros1-practice.git
$ cd ros1-practice/
$ cp -r  hello_world ~/catkin_ws/src/
$ catkin build
```
を実行してください
